# Default Artist Portfolio Template

This is a neutral, minimal HTML/CSS/JS portfolio base.
You can open `index.html` directly in your browser.

## Structure
- index.html — gallery page
- about.html — about page
- writing.html — writing page
- style.css — controls color, font, layout
- script.js — controls image placement

## How to customize
- Change your name and text in the HTML files.
- Replace images in the `assets/` folder (create if missing).
- Adjust colors or fonts in `style.css`.
- Add or remove works in the `items` array in `script.js`.

## Publish
Upload all files to a GitHub repo and enable GitHub Pages.
